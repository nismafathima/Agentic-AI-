## ğŸš€ Usage

### 1. Upload a PDF
```python
from google.colab import files
uploaded = files.upload()
```

### 2. Load and Split Document
```python
from langchain_community.document_loaders import PyPDFLoader
from langchain_text_splitters import RecursiveCharacterTextSplitter

loader = PyPDFLoader("/content/your_file.pdf")
document = loader.load()

splitter = RecursiveCharacterTextSplitter(chunk_size=500, chunk_overlap=50)
chunks = splitter.split_documents(document)
```

### 3. Create Embeddings and Vector Store
```python
from langchain_huggingface import HuggingFaceEmbeddings
from langchain_community.vectorstores import FAISS

embeddings = HuggingFaceEmbeddings(model="sentence-transformers/all-MiniLM-L6-v2")
vector_db = FAISS.from_documents(chunks, embeddings)
```

### 4. Load Language Model
```python
from transformers import AutoTokenizer, AutoModelForCausalLM, pipeline
from langchain_community.llms import HuggingFacePipeline

tokenizer = AutoTokenizer.from_pretrained("gpt2")
model = AutoModelForCausalLM.from_pretrained("gpt2")

pipe = pipeline("text-generation", model=model, tokenizer=tokenizer, max_new_tokens=60, temperature=0.7)
llm = HuggingFacePipeline(pipeline=pipe)
```

### 5. Agentic AI Q&A
```python
def agentic_pdf_assistant(question):
    # Step 1: Retrieve context
    docs = vector_db.similarity_search(question, k=3)
    context = "".join([doc.page_content for doc in docs])

    # Step 2: Agent decides how to answer
    prompt = f"""
    You are an Agentic AI assistant. Use the context below to answer the question.
    If context is insufficient, reason and explain clearly.

    Context:
    {context}

    Question:
    {question}

    Answer:
    """
    return llm.invoke(prompt)
```

### 6. Interactive Mode
```python
print("ğŸ“„ PDF RAG Assistant with Agentic AI")
print("Type your question below. Type 'exit' to quit.\n")

while True:
    user_question = input("You: ")
    if user_question.lower() == "exit":
        print("\nğŸ¤ Thank you for using PDF RAG Assistant.")
        break
    response = agentic_pdf_assistant(user_question)
    print("\nAssistant:", response, "\n")
```
